Вы руководите музыкальным коллективом, который использует следующие инструменты: гитару (Guitar), барабаны (Drums), пианино (Piano), флейту (Flute), которые издают соответствующие звуки: ‘guitar music’, ‘drums music’, ‘piano music’, ‘flute music’. Иногда инструменты ломаются и их приходится отдавать в ремонт, однако, репетиции все равно должны продолжаться. Именно для этих целей вы смогли найти специальный универсальный электронный инструмент (digital), который умеет как издавать собственные звуки (‘digital music’), так и подражать другим музыкальным инструментам.
Ваша задача - реализовать классы для каждого музыкального инструмента, а также адаптеры, чтобы электронный инструмент мог издавать те же звуки, что и другие инструменты.
В этой миссии вам может помочь такой шаблон проектирования, как Adapter.

Примеры: 

Guitar().play_guitar() == "guitar music"
Drums().play_drums() == "drums music"
Piano().play_piano() == "piano music"
Flute().play_flute() == "flute music"
Digital().play_digital() == "digital music"
GuitarAdapter().play_digital() == "guitar music"
DrumsAdapter().play_digital() == "drums music"
PianoAdapter().play_digital() == "piano music"
FluteAdapter().play_digital() == "flute music"


Входные данные: методы классов музыкальных инструментов и адаптеров.

Выходные данные: правильная музыка.

Как это используется: Очередной шаблон проектирования.

Предусловие: Все данные корректны.
